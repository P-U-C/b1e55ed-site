An open-source AI trading intelligence that anyone can run. Clone it, bring your own API keys, pick a preset, start paper trading in minutes.

The system compounds its own learning — every trade, every pattern, every mistake feeds back into a grimoire that gets sharper over time. Infinitely configurable for builders who want to tune every weight.

Not a bot that buys and sells. A signal → decision → learning pipeline where the next cycle is always smarter than the last.

What's inside
  • 7-domain synthesis engine (on-chain, TradFi, sentiment, social, technical, events, curator)
  • Regime detection with real features (stablecoin flows, ETF data, funding z-score)
  • Conviction scoring with automatic counter-thesis when confidence runs too high
  • Event-sourced database with hash chain (tamper-evident, replayable)
  • 5-level kill switch hierarchy (pause → reduce → exit → lockdown → emergency)
  • Config presets: conservative, balanced, degen — or build your own
  • Backtest engine with walk-forward validation and FDR correction
  • Mandatory paper trading period before live execution
Pipeline
Producers → Event Store → Normalizer → Synthesis (7 domains) ↓ Feature Extractor → Regime Detector → Conviction Engine ↓ Kill Switch → Decision Engine → Alerts / Actions ↕ ↕ Grimoire Hooks (read patterns pre-synthesis, write outcomes post-decision)
